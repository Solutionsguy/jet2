## 1. **Overview**
The multiplier glow effect is a **visual enhancement** that dynamically changes the color and glow intensity of the multiplier text as the game progresses. It provides players with **real-time feedback** on risk and reward escalation, improving engagement and decision-making.

---

## 2. **Objectives**
- **Enhance Visibility**: Make the multiplier the most prominent element on screen.  
- **Signal Risk/Reward**: Use color progression to indicate increasing volatility.  
- **Build Excitement**: Intensify glow and color transitions to heighten tension.  
- **Improve UX**: Provide intuitive visual cues without requiring players to read exact numbers.  

---

## 3. **Functional Requirements**
- **Real-Time Updates**: Multiplier value and glow must update continuously.  
- **Color Progression**: Colors shift based on multiplier thresholds.  
- **Glow Animation**: Outer glow intensity increases with multiplier.  
- **Smooth Transitions**: Color and glow changes must animate smoothly (CSS transitions).  
- **Responsive Design**: Effect must scale across desktop and mobile.

### 5.2 JavaScript Logic
```javascript
function updateMultiplier(multiplier) {
  const el = document.getElementById("multiplier");
  el.textContent = multiplier.toFixed(2) + "x";
  el.className = ""; // reset classes

  if (multiplier < 2) el.classList.add("blue");
  else if (multiplier < 10) el.classList.add("green");
  else if (multiplier < 50) el.classList.add("orange");
  else if (multiplier < 100) el.classList.add("red");
  else el.classList.add("purple");
}
```
